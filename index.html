About Coding

What is HTML?

HTML stands for HyperText Markup Language. HTML is the basic structure of a website. You can add all sorts of things using HTML code!

Including:

  • Titles (what displays on the tab)
  • Headings 1-6 for size
  • Images and links
  • Video and audio
  • Forms and buttons
  • Numbered and bulleted lists
  • Paragraphs
  • Basic Styling
  • Accessibility
What is CSS?

CSS stands for Cascading Style Sheets. It's the color and styling of the website.

How-to Steps

  1. Add the doc type and opening and closing HTML
  2. Add a head section and body section. Add a title and meta information in the head section. (Always include charset, preferably UTF-8)
  3. Add heading in the body section, along with elements like lists, paragraphs, images, and so on.
  4. Add things to help with accessibility.
    Examples:
    • strong and emphasis tags
    • You can also change the color and other aspects
    • alt for img description
    • footer, main and section tags
    • address
      and time tags
    • abbr and so on.
  5. Add styling and start using some CSS aspects for details.

Tips/tricks

  • You can put the br tag for a line break
